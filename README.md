## 📖 About This Repository

This repository is my **daily Python practice log** — a collection of Jupyter Notebooks documenting my journey from absolute basics to more advanced concepts like Object-Oriented Programming and file handling. Each notebook is hands-on, code-first, and built for learning by doing.

> 💡 *"Consistency beats intensity — one notebook a day adds up to mastery."*
